# Benchmarks

This project includes performance benchmarks for critical application operations using [BenchmarkDotNet](https://benchmarkdotnet.org/).

## Running Benchmarks

To run the benchmarks, navigate to the `benchmarks/aspnet-spa-template.Benchmarks` directory and execute:

```bash
dotnet run -c Release
```
